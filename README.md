## ğŸ’¡ Quick Examples

**Factorial:**
```python
# factorial.py
def factorial(n):
    return 1 if n == 0 else n * factorial(n - 1)

print(factorial(5))  # Output: 120
```

**OOP â€” Inheritance:**
```python
# inheritance.py
class Animal:
    def speak(self):
        print("Animal speaks")

class Dog(Animal):
    def speak(self):
        print("Dog barks")

d = Dog()
d.speak()  # Output: Dog barks
```

**Lambda + Filter:**
```python
# filter_map_reduce.py
nums = [1, 2, 3, 4, 5, 6]
evens = list(filter(lambda x: x % 2 == 0, nums))
print(evens)  # Output: [2, 4, 6]
```
